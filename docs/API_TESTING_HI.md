# एग्रीनेट API परीक्षण गाइड (Agrinet API Testing Guide)

यह गाइड curl कमांड का उपयोग करके Chat-UI घटकों के लिए बैकएंड API को परीक्षण करने में आपकी मदद करेगी।

## 1. सेवा पोर्ट्स और एंडपॉइंट्स

| सेवा | पोर्ट | उपयोग |
|------|-------|--------|
| बैकएंड | 5000 | API अनुरोध |
| फ्रंटएंड | 3000 | Next.js UI |

## 2. प्रमाणीकरण

अधिकांश API एंडपॉइंट्स को एक वैध API कुंजी की आवश्यकता है। हेडर के साथ पास करें: `x-api-key`

## 3. मुख्य शब्दावली

- **बैकएंड** - Backend: सर्वर-साइड API
- - **फ्रंटएंड** - Frontend: उपयोगकर्ता इंटरफेस
  - - **कुंजी** - Key: प्रमाणीकरण टोकन
    - - **त्रुटि** - Error: अनुरोध में समस्या
      - - **एंडपॉइंट** - Endpoint: API का URL पता
       
        - ## 4. महत्वपूर्ण कमांड्स
       
        - ```bash
          # स्वास्थ्य जांच करें
          curl -X GET http://localhost:5000/health

          # बातचीत बनाएं
          curl -X POST http://localhost:5000/conversations

          # संदेश भेजें
          curl -X POST http://localhost:5000/messages/<id>
          ```

          ## 5. सामान्य त्रुटियां

          | त्रुटि | अर्थ |
          |-------|------|
          | Cannot GET | गलत पोर्ट या एंडपॉइंट |
          | Unauthorized | अमान्य API कुंजी |
          | 404 Error | पृष्ठ नहीं मिला |

          ## 6. समस्या निवारण

          - पोर्ट 5000 पर बैकएंड जांचें
          - - API कुंजी सत्यापित करें
            - - बैकएंड लॉग देखें
